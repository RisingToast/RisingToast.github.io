- 잡다한 깃허브 블로그

## 웹사이트 주제
- 내 관심사와 관련된 다양한 카테고리별 블로그
- 주제 : 게임, 일상, 코딩관련지식

## 프로젝트 목표:
- 1. 최소 3개 이상의 카테고리만들기
- 2. 정확한 정보와 적당한 글과 사진을 추가해서 만들기
- 3. 과제 제출 후에도 사용가능한 블로그 만들기

## 대상(타겟층, 고객층) 
- 관심사가 겹치는 사람
- 20대, 30대 대학생, 게이머, 전공자 등
